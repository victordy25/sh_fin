자금 현황
내가 실제로 쓸 수 있는 돈의 전체 구조를 파악하는 섹션입니다
유동성 자산
단위: 만원
자산 파악용
입력 예시
20,000
현금, 예금, 적금, 주식·펀드처럼 비교적 빠르게 현금으로 바꿀 수 있는 자산의 합계입니다. 부동산·자동차처럼 바로 팔기 어려운 자산은 제외합니다.
은행 앱 → 총 잔액 / 증권사 앱 → 평가금액 합산 / 국세청 홈택스 → 금융정보 조회
입력 방법
예금 8,000 + 적금 3,000 + 주식 평가액 5,000 + 현금 4,000 = 2억원 → 20,000 입력
주식은 현재 평가금액 기준 (매수가 아닌 현재 시세)으로 입력하세요
연금저축·IRP 등 중도해지 가능한 연금도 일부 포함할 수 있지만, 세금·패널티를 고려해 별도 연금 자산으로 분리 입력하는 것이 더 정확합니다

국민연금 → 내연금 알아보기(nps.or.kr) / 퇴직연금 → 각 증권사·은행 앱 / 개인연금 → 금융사 앱
입력 방법
연금저축 잔액 2,000 + IRP 1,500 + 퇴직금 예상 1,500 = 5,000 입력
국민연금은 월 수령액이 아닌 지금까지 납입한 원금 기준으로 넣어도 됩니다

단위: 만원
가장 중요즉시 현금화 가능
입력 예시
15,000
지금 당장, 올해 안에 현금화해서 주거 비용으로 쓸 수 있는 금액입니다. 유동성 자산에서 당장 팔기 어려운 자산을 빼거나, 실제 통장에 있는 돈 기준으로 입력합니다. 시뮬레이션의 초기 투자금 기준이 됩니다.
입력 방법

손실 위험 있는 주식은 현재 평가금액의 80~90%만 반영하는 것을 권장합니다

신용대출 / 주택담보대출(기존) / 기타 대출

은행 앱 → 대출 현황 / 금융결제원 어카운트인포(accountinfo.or.kr) → 대출 내역 일괄조회

비상예비금 목표
단위: 만원 | 권장: 월생활비 × 3~6개월
유동성 완충
입력 예시
3,000
실직·의료비 등 비상상황을 위해 주거 자금과 별개로 반드시 남겨두어야 하는 현금입니다. 이 금액은 가용 자금에서 먼저 제외하고 시뮬레이션을 진행합니다.
입력 방법
월 생활비 300만원 × 6개월 = 1,800 ~ 3,000 입력
외벌이 가구는 6개월치, 맞벌이는 3개월치를 권장합니다
DSR 한도 / LTV 한도
단위: % | 현재 규제 기준값 입력
금융 규제
2025년 기준
DSR 40% / LTV 70%
DSR(총부채원리금상환비율): 연소득 대비 모든 대출의 연간 원리금 합계 비율. 이 비율 이내에서만 대출이 가능합니다.

LTV(담보인정비율): 아파트 시세 대비 최대 대출 가능 비율입니다.
금융감독원 금융소비자포털(fine.fss.or.kr) → 주택담보대출 규제 현황 / 은행 상담
투기과열지구·조정대상지역은 LTV가 더 낮게 적용됩니다 (40~50%). 해당 지역 아파트라면 LTV를 낮춰서 입력하세요
DSR 계산: (연간 원리금 합계 ÷ 연소득) × 100
예) 월 원리금 150만원 × 12 = 1,800만원 ÷ 연소득 6,000만원 = 30% → 한도 내
